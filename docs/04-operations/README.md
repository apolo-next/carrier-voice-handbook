## On war stories

`common-failures.md` is, in my experience, the single most useful file
for a working engineer. The handbook actively solicits war stories from
contributors — see [CONTRIBUTING.md](../../CONTRIBUTING.md) for the
template. A short story (200-300 words) about one real incident teaches
more than a chapter of theory.
