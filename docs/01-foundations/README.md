## Reading order

Read top to bottom. Each builds on the previous.

If you have voice experience already on a proprietary stack
(Cisco, Ribbon, Sonus, Genband), `why-open-source-carrier.md` is the
most useful entry point — it frames why this handbook exists at all
and what changes when you cross over.

If you have networking experience but no voice, start with
`sip-mental-model.md` first. The "why open source" question is less
relevant if you don't yet know what an alternative looks like.
