## Philosophy

These roadmaps assume:

- You have basic networking fluency (TCP/IP, routing, firewalls).
- You can read code and write small scripts.
- You have access to a Linux box where you can break things.

They do *not* assume:

- A computer science degree.
- Prior voice or telecom experience.
- Access to commercial vendor training.

The goal is for someone with discipline and a laptop to reach a
mid-senior level over 18-24 months of deliberate practice.
